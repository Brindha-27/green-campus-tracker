## 📖 About the Project

Green Campus Tracker is a web-based application designed to promote sustainable practices among students and individuals within a campus environment. The application allows users to record, monitor, and reflect on their daily eco-friendly actions.

The main goal of this project is to encourage users to adopt environmentally responsible habits such as saving electricity, reducing plastic usage, conserving water, and promoting green living.

---

## ⚙️ Features

- ✅ Add eco-friendly actions performed daily  
- ❌ Remove actions by clicking on them  
- 🔢 Dynamic counter to track total contributions  
- 💡 Random eco tips to encourage sustainable behavior  
- 💾 Data persistence using localStorage (data stays even after refresh)

---

## 💡 How It Works

Users can enter any eco-friendly activity they perform, such as turning off unused lights or using reusable materials. These actions are displayed in a list and counted to provide a sense of contribution.

The eco tip feature provides simple and actionable suggestions that users can follow in their daily life. This makes the application not only a tracking tool but also an awareness platform.
